# 🧩 Arquitetura do Connection Cyber OS 2.0

A arquitetura é modular, escalável e alinhada a padrões enterprise.

---

## 🖥️ Frontend

- Next.js 14 (App Router)
- Server Actions
- Tailwind CSS
- Design Tokens
- Componentes acessíveis (WCAG)

---

## 🛠️ Backend

- Supabase Auth
- RLS (Row Level Security)
- RBAC (Roles)
- Policies
- Edge Functions

---

## 🗄️ Banco de Dados

- PostgreSQL (Supabase)
- Prisma 7
- Migrations
- Auditoria
